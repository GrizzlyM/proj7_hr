В этом проекте я выступил в качестве hr-аналитика, помогающего бизнесу оптимизировать работу с персоналом: бизнес предоставляет данные, 
а аналитики предлагают, как избежать финансовых потерь и оттока сотрудников. Основными моими задачами были: обучение как минимум двух моделей. 
Взял одну линейную модель для предсказания уровня удовлетворенности сотрудников от 0 до 1, а в качестве второй — дерево решений для предсказания ухода сотрудника. 
Подберите гиперпараметры как минимум для одной модели с помощью одного из известных вам инструментов.
