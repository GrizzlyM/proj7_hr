В этом проекте я выступил в качестве hr-аналитика, помогающего бизнесу оптимизировать работу с персоналом: бизнес предоставляет данные, 
а аналитики предлагают, как избежать финансовых потерь и оттока сотрудников. Основными моими задачами были: обучение минимум двух моделей. 
Отобрал лучшую линейную модель для предсказания уровня удовлетворенности сотрудников от 0 до 1, а в качестве второй — дерево решений для предсказания ухода сотрудника. 
Осущетсвил подбор гиперпараметров для одной модели с помощью одного из известных инструментов.
